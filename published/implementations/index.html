eBraille Implementation Report

About this document

This document lists known implementations of the DAISY ebraille specification.

Implementations fall into one of three categories:

  • Validators
  • Reading systems
  • Authoring tools

To add an implementation to the report, please open a pull request in the eBraille GitHub repository.

Implementations

Validators

eBraille Checker

Based on EPUBCheck, the eBraille Checker validates eBraille files for conformance to the standard.

Currently in beta release.

Reading Systems

Monarch

eBraille files can be loaded and read on Monarch refreshable braille devices.

Authoring Tools

brf2ebrl

brf2ebrl is a transformation script that converts brf digital braille files to eBraille.

Currently in beta release.

DAISY Pipeline 2

The Pipeline 2 application transforms DAISY 3 DTBook files to eBraille.

Currently in beta release.
